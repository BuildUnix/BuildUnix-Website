# Setup Order

If you want to recreate or paste this project manually in VS Code, use this order:

1. `package.json`
2. `next.config.mjs`
3. `jsconfig.json`
4. `app/layout.js`
5. `app/globals.css`
6. `lib/site-content.js`
7. `lib/seo.js`
8. `hooks/usePrefersReducedMotion.js`
9. `components/SmoothScrollProvider.jsx`
10. `components/SplitTextReveal.jsx`
11. `components/PageTransition.jsx`
12. `components/PageTransition.module.css`
13. `components/SiteHeader.jsx`
14. `components/SiteHeader.module.css`
15. `components/SiteFooter.jsx`
16. `components/SiteFooter.module.css`
17. `components/Reveal.jsx`
18. `components/Reveal.module.css`
19. `components/HeroMorph.jsx`
20. `components/HeroMorph.module.css`
21. `components/HeroMedia.jsx`
22. `components/HeroMedia.module.css`
23. `components/CountUp.jsx`
24. `components/DemoForm.jsx`
25. `components/DemoForm.module.css`
26. `components/home/HomeExperience.jsx`
27. `components/about/AboutExperience.jsx`
28. `components/contact/ContactExperience.jsx`
29. `app/page.js`
30. `app/home.module.css`
31. `app/about/page.js`
32. `app/about/about.module.css`
33. `app/contact/page.js`
34. `app/contact/contact.module.css`
35. `app/robots.js`
36. `app/sitemap.js`
37. `public/og/buildunix-og.svg`
38. `public/brand/buildunix-logo.png`
39. `app/favicon.ico`
40. Upload the optional media files listed in `public/media/README.md`

## Run locally

1. `npm install`
2. `npm run dev`

## Production build

1. `npm run build`
2. `npm run start`
